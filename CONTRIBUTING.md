# Contributing to LoCoPilot

Welcome, and thank you for your interest in contributing to LoCoPilot!

LoCoPilot is a code editor developed by **Bixbite AI**. There are several ways you can contribute.

## Asking Questions

Have a question? Open a [Discussion](https://github.com/BixBiteAi/LoCoPilot/discussions) on GitHub, or file an issue with the "question" label. For general editor usage questions, [Stack Overflow](https://stackoverflow.com/questions/tagged/visual-studio-code) is also a good resource.

## Reporting Issues

Found a bug or have a feature request for LoCoPilot?

1. **Search first:** Check [open issues](https://github.com/BixBiteAi/LoCoPilot/issues) to see if it's already reported.
2. **File an issue:** Use the appropriate [issue template](https://github.com/BixBiteAi/LoCoPilot/issues/new) (bug report or feature request).
3. **Include:** LoCoPilot version, OS, steps to reproduce, and what you expected vs. what happened.

You can also use **Help > Report Issue** in LoCoPilot to pre-fill version and environment details.

If the issue seems to come from upstream (and not LoCoPilot-specific), you may report it to the [upstream project](https://github.com/microsoft/vscode/issues) as well.

## Contributing Code

1. **Fork** the [LoCoPilot repository](https://github.com/BixBiteAi/LoCoPilot) and clone your fork.
2. Create a **branch** for your change (e.g. `fix/issue-123` or `feat/my-feature`).
3. Make your changes and ensure tests pass where applicable.
4. Open a **Pull Request** against the main branch of `BixBiteAi/LoCoPilot`. Describe what you changed and reference any related issues.

We will review your PR and may request changes. Once approved, it can be merged.

## Thank You

Your contributions make this project better. Thank you for taking the time to contribute.
