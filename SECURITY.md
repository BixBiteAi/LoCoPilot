## Security

Bixbite AI takes the security of LoCoPilot seriously.

**Please do not report security vulnerabilities through public GitHub issues.**

If you believe you have found a security vulnerability in LoCoPilot, please report it responsibly:

- **Email:** Report to Bixbite AI through your usual contact channel, or open a private security advisory on GitHub.
- **GitHub:** You can use [GitHub's private vulnerability reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/about-github-security-advisories) for this repository (enable it in the repository settings under Security > Code security and analysis).

We will acknowledge your report and work with you to understand and address the issue.
